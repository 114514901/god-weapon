# God Weapon

Forge 1.20.1 多功能武器模组

## 功能

- **右键** — 清除 16 格内所有生物实体（无视血量/免疫/模组保护）
- **H 键**（手持时）— 打开能力 GUI
- **无敌模式** — 物品在背包即生效
- **飞行模式** — 物品在背包即生效
- **夜视效果** — 物品在背包即生效

## 获取

```
/give @p godweapon:god_weapon
```

## 构建

```bash
./gradlew build
```

产物在 `build/libs/`。需要 JDK 17。
