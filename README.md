# 氪金萝莉-重生

Forge 1.20.1 多功能武器模组

## 物品

### 氪金萝莉 (`loli_reborn:loli_pickaxe`)

右键清除实体，左键范围挖掘，H 键打开能力 GUI。

| 能力 | 说明 |
|------|------|
| 无敌 | `LivingHurtEvent` + Mixin `Entity#remove` 双重拦截 |
| 飞行 | 背包内生效 |
| 夜视 | 背包内自动续药水 |
| 急迫+力量 | 背包内自动续药水 |
| 反伤+吸血 | 无敌开启时攻击者反弹伤害，玩家回复 30% 攻击力 |
| 摔落免疫 | 飞行开启时自动取消 |
| 范围挖掘 | 左键触发，分批处理不卡服 |
| 生物排斥 | 8 格内生物每 tick 外推 |
| 清除实体 | 先伤害拿掉落，死不掉的强制移除 |

### 普通萝莉 (`loli_reborn:basic_loli`)

无限耐久多功能工具（镐+斧+锹），可升级为氪金萝莉。

## 合成

- **普通萝莉**：铁斧 + 铁镐 + 铁锹 + 2 铁锭
- **氪金萝莉**：4 下界之星 + 普通萝莉

## 构建

```bash
./gradlew build
```

JDK 17，产物在 `build/libs/`。
